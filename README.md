# 📅 Event Calendar

```
A dynamic interactive calendar web app built with React. Users can view events, see event descriptions, and navigate through different years. Perfect for tracking schedules, holidays, or custom events.
```
## 🔹 Features
```
🗓 Interactive Calendar – Browse months and select dates
⭐ Event Highlights – Dates with events are visually marked
📝 Event Details – Click on a date to see event description
🔄 Year Navigation – Switch between years to check past or upcoming events
🎨 Responsive Design – Works on mobile, tablet, and desktop

```
## 💻 Tech Stack
```
Vue – Front-end library
CSS / Styled Components – Modern styling
Optional: Local storage or JSON data for events

```
## ⚡ Usage / Installation
```
1. Clone the repository:
git clone https://github.com/darshiu/calendar.git
cd calendar
2. Install dependencies:
npm install
3. Start the development server:
npm run dev
4. Open the app in your browser:
http://localhost:5173
```
